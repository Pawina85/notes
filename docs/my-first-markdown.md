# My First Markdown Guide

Welcome to your first markdown document! This file shows you the basics.

## Text Formatting

You can make text **bold** or *italic*.

You can also combine them: ***bold and italic***.

## Lists

### Unordered Lists
- First item
- Second item
  - Nested item
  - Another nested item

### Ordered Lists
1. First step
2. Second step
3. Third step

## Code Examples

Here's some inline code: `console.log('Hello World')`

Here's a code block:

```javascript
function greet(name) {
  return `Hello, ${name}!`;
}

console.log(greet('Docusaurus'));
```
